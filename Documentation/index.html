Uploading

After you chose a page that suitable you, for the right work necessary to upload it (this also applies to the builder). To do it you can use online FTP the manager of your hosting (almost all the hosting companys has FAQ/support pages where they explain how to upload files via their own FTP managers) or use the desktop version (FileZilla for example).

  1. Find your FTP Settings. In order to use FTP with your hosting, you will need the following settings from your hosting account. Host, Username and Password.
  2. Upload the chosen files. Now when you know your account settings (host, username and password) you just need connect to your server via FTP client (we recommend FileZilla, video tutorial) and upload files.
  3. Once the files are done uploading, you can try access your website via browser.
Performance of this operation means that you have the server space and the domain.

Page builder

What it is Page Builder is a Javascript script which allows publishers of HTML templates to provide their customers with a quick and easy way to customise HTML templates. Page Builder runs (almost) entirely in the browser. The only server side code is in the form of a single PHP file used for exporting HTML markup.


What it's not (!) Page Builder is NOT a CMS (content management system). It won't work with WordPress. It does not provide a way for end-users to work on live websites and make changes to such websites.

How it works

Meet the SUPRA, new modern page builder that helps you to create your own optimized HTML5 page in a minutes without any knowledge of code. This tool is very easy to install and use. Just follow steps below:

  1. Download archive with the builder from the ThemeForest.
  2. UnZip archive.
  3. Upload folder with the builder ("Pagebuilder" folder) to the server via FTP.
  4. Once uploaded, point your browser to said server/hosting and your script will work.
  5. Create and download your own HTML5 landing pages.
Requires:
  • PHP v5.3 (not less).
  • Apache webserver (Windows servers will probably work, but are not supported).
  • An FTP tool to upload the files.

Add section

Choose "Sections" mode. Page builder allows users to drag sections onto a canvas and drop them where they'd like them to appear. Once dropped onto the canvas, the position can be changed by dragging and dropping as well. Sections can be delete individually.

Section settings

Choose "Sections" mode. To see section settings just hover on it. You can move, change background, setup advanced settings, copy and delete section.

Dafault styles

Choose "Elements" mode. Default styles are divided into styles for light skin and dark. You can choose skin for each section in advanced section settings.

Edit elements

Choose "Elements" mode. Hover on the element to see available settings. Some elements have more settings, some less. Starting from version 2 you can add animation and vertical margins to elements.

Edit text

Choose "Typography" mode. In the left panel you can choose font family, font size and font styles for whole page. This option available for default font and header texts. Highlight some text to see the editor, if you want highlight all block - press (CMD + A) or (CTRL + A). You can edit any text that you see in the builder's work frame.

Edit icons

Choose "Typography" mode. In SUPRA we have included more than 1,000 icons. Just hover on the icon and choose "icons list" to see them all. Select needed icon by clicking on it and press "Apply" button.

Page settings

Choose "Project" mode. In the left panel press the settings icon that next to the page name. Page settings include: General settings, SEO and Preloader settings. As well you can delete page (except index), create new or duplicate it.

Download/Save/Load project

Choose "Project" mode. In the left bottom corner you can see project options. If you want create new project don't forget to save old. Projects are saved in *.supra format. If you download ready page there will be no unneeded pictures or scripts, all data optimized.

After creating new, current project will be deleted. Don't forget to save the project.

Create new section

The best way to create a section - find a similar (in already existing) as a base and copy it. All sections should have a unique name/ID. So, you need to change: section ID in the html code, folder name, thumbnail name (in "images" folder). All of them should be named the same and don't forget to change IDs in style.css and paths for images in the html code.

Congrats! Now you have a platform for your new section. To modify the html code of the section use index.html (located in section folder). If you want change/add CSS code - use style.css (located in section folder). For javascript - use custom.js (located in section folder). If there is no style.css or custom.js - you have to create them yourself.

You can change the order of sections (and groups) in the left sidebar in order.xml.

Add new font

To add a new Google font, open the file js/fonts.js and insert your desired font. Each line specify only base and bold value for current font (italic can be used too).

Add new icons

All icons used by the builder are stored in the css/icons.css file. To add new icons you need to describe them in this file. The class name must begin with the prefix "icon-" (example: icon-pen, icon-home, icon-paint-roller, etc). Icon names can't be repeated, they all should be unique. Sources of icons you have to add to the basic icon font (iconfont) in the "fonts" folder. Best way to do this - use icomoon service (as we do).

Ready templates

This set contains ready templates for different needs. These templates are simple HTML5 pages and you can upload them on your server/hosting via FTP manager. All of these templates were based on our new page builder (SUPRA). So you can upload any of ready templates into the builder and customize it like you want (use *.supra project file for uploading). If you decide to customize ready templates without builder, we'll try to help with this. Here some points for you:

Sections

Each section include few option-classes. You can customize sections by changing these classes.

  • pt-125 - padding top 125px (values: 0,25,50,75,100,125,150,200,250).
  • pb-150 - padding bottom 150px (values: same as top).
  • md-pb-150 or md-pt-150 - padding that reset if the width is less than 992px.
  • dark - section skin (values: light or dark, light by default).
  • bg-3-color-dark - background color ( values: bg-1-color-light, bg-2-color-light, bg-3-color-light, bg-1-color-dark, bg-2-color-dark, bg-3-color-dark).
  • parallax-bg - add parallax effect to background.
<section id="section-name" class="pt-125 pb-150 bg-3-color-dark dark">
    <div class="container">
        <div class="row">
            <div class="col-md-12">
                SECTION CONTENT
            </div>
        </div>
    </div>
    <div class="bg parallax-bg"></div>
</section>

Icons

All 1000+ icons are embedded in a webfont for more convenient use on the web or in a native application. To change an icon to one you need - just change a class of this icon.

  • icon-color - add special icon color (by default color same as font color).
  • icon-position-left - add margin right 10px (values: icon-position-right, icon-position-left).
  • icon-size-m - icon size (values: icon-size-m, icon-size-l, icon-size-xl).
  • icon-plus - icon name (view all icons here).
<i class="icon-color icon-position-left icon-size-m icon-plus"></i>

Contact form

In order to add your email address (and subject text) to contact form, need to open "scripts/request.php" and change code, see example:

if($_POST['id'] === "from-id") {
    $to = "paste your email address here"; // Your e-mail address here.
    $body = "\nName: {$_POST['contactname']}\nEmail: {$_POST['contactemail']}\n\n\n{$_POST['contactmessage']}\n\n";
   mail($to, "paste your subject text here", $body, "From: {$_POST['contactemail']}"); // E-Mail subject here.
}
